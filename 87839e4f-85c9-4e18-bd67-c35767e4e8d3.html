Description

Langton's Ant at its heart is a simple set of instructions which can result in amazing patterns; or just nonsense movements.

Langton's Ant is also a 2D universal Turing machine and was invented by Chris Langton in 1986.
This allows the community to discover patterns which the world has never seen before!

My version, in addition to the base ruleset, includes the ability to input more than one of the same color with different commands which allows for some really interesting pattern generation of which has never been seen before!
