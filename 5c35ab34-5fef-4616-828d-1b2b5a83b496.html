Description

Endless Combat, is a coop game which is about fighting as characters with unique skills against zombies, the game also contains free for all mode; We used UNREAL ENGINE 4 game engine to create a better network system,effects,game mechanics and maps.

New Network System

  • You can register and access the game world.
  • You can buy weapons from shop menu in mainmenu
  • You can change your character and ultimate whenever you want

Coop Mode Arrived

  • Before play coop mode we suggest buy a weapon via shop!
  • You can play solo or with online players.
  • It has 3 difficulty levels

You can create lobby and play with friends or join somebodies lobby and hangout with them.

About Game Development

Endless Combat created by indie game studio Fatalitech and our first game.
