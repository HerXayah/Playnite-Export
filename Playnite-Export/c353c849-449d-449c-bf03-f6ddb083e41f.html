Description

The long-lost classic puzzle from Windows XP Plus! - Package lives again here.
Like the original Microsoft Russian Square Plus! Edition you have to proof your ability to create matching lines of blocks before time runs out.
90s feeling repackaged. Cool music and sounds accompany you through the different game modes.
Numerous simple but also more demanding achievements complete the gaming experience.
Compete with others in Square Games Leaderboard

Gameplay:
  • Use the keyboard, mouse or touchdevice to move the active block around and push it into the Square so that all the blocks in a row or column match. When the blocks match , the row or column disappears.

  • When all blocks are eliminated, you move up to the next level.
  • The higher the level of difficulty, the shorter the time you have to solve the level.

Game modes:
  • Classic: 9 Level - like back then
  • Arcarde: endless mode
