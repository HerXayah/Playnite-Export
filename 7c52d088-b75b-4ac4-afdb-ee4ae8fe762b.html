Description

a simple but difficult puzzle game
Dragons lived.
One of them touched the devil's books and all the other dragons were infected.
But only the just-born baby "Yongyong" was not infected.
Can Baby Dragon Save the World?
Baby dragons lack mana, so they have to collect all the stars (horsepower) with minimal movement.
Use WASD to control the baby dragon that goes before it hits the wall, eat all the stars, and defeat the devil.
Try clearing away from all the traps!

Challenge the brainiest puzzle imaginable!
