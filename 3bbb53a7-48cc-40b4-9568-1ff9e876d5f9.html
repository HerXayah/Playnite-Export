Description

In fact, a short description of this game has deceived you.
You will be required not only to be attentive, but also to react in a timely manner, to have intelligence and a good memory.
But all this is useless without mindfulness, so that`s the key.



ITRP _ Gas Prison will allow you to arrange a little test.
You will find yourself in a system of cells, each of which can slam and poison you with gas.
Keep an eye on the indicators to prevent this.
However, the conditions under which you will be placed will not allow you to save all cells. Some will have to be sacrificed. And there are few useless ones among them:
  • Some contain maps or timers.
  • Some stores interactive monitors for system management.
  • Some just has a lot of passages.

The game is NOT designed to test your reflexes, reaction, accuracy or ...patience.
It is designed to test your attentiveness, intelligence, ability to think ahead.

Even if you need time to get used to and remember the location of the functional points, nevertheless, you will learn the detailed rules inside.
