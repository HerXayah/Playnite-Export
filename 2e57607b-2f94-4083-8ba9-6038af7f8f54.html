Description

5 vs 5 multiplay shooting game.

Using various heroes with different abilities, you can play various roles in battlefields such as 'Assault', 'Deathmatch', 'Escort', 'Battle Royale' and ETC.
Each hero can be raised individually, they also have own 'ranking', 'weapons' and 'skill'.


==== Features ====
- Easy to Control
- Small game size
- Low latency online FPS
- Low specification game

1. In-game guide, please use the book button on the top left corner.
2. If you use Hack or Patcher, you can be blocked permanently.
