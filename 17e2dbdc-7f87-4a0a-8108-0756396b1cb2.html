Case Kovacs - Agent 228

... is a prelude to the game NoseBound. This very short and intriguing mini game tell the story of Elizabeth Kovacs, the missing woman from the story NoseBound.

Please consider the fact that we made this game mostly to test gameplay mechanics in Unreal Engine, exploring new tools, etc. The gameplay, although still point & click, is very different.

https://store.steampowered.com/app/340020/NoseBound/

There is a possibility to extend this title onto a fully developed game once NoseBound is released if there's enough interest.
