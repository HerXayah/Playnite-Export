Description

cameleon is an application that adjusts camera images and displays them on various applications through virtual device drivers.
It was developed in such a way that even if the location of shooting, the type of lighting equipment used, or the location of the arrangement of the lighting equipment changes, the image can be adjusted to deliver a stable expression/atmosphere by adjusting the image.
We think this application can be used in situations such as web conferences, web interviews, and live streaming.

Functions

  • Filter adjustment
  • Brightness, color adjustment
  • Save multiple settings
  • Place image (layer) freely
  • Input text (layer) freely
